## Capture Instructions

### For Mobile App Screenshots:
1. Open the mobile app on your device or emulator
2. Navigate to each screen listed above
3. Take a screenshot:
   - **iOS**: Press Volume Up + Power button simultaneously
   - **Android**: Press Volume Down + Power button simultaneously
   - **Emulator**: Use the emulator's screenshot tool or device screenshot
4. Save screenshots to: `docs/how-to-guides/images/screenshots/mobile-app/`
5. Name them exactly as listed above

### For Web Portal Screenshots:
1. Open the web portal in Chrome or Firefox at 100% zoom
2. Set browser window to 1920x1080 resolution
3. Navigate to each screen/feature
4. Take a screenshot:
   - **Windows**: Use Snipping Tool or Win+Shift+S
   - **Mac**: Cmd+Shift+4 for area selection
   - **Browser**: Use browser extensions like "Full Page Screen Capture"
5. Save screenshots to: `docs/how-to-guides/images/screenshots/web-portal/`
6. Name them exactly as listed above

## Tips for Good Screenshots

1. **Use sample/test data** - Don't use real sensitive information
2. **Consistent sizing** - Keep browser window size consistent
3. **Clear text** - Ensure all text is readable
4. **Full context** - Show enough of the interface to understand the feature
5. **Clean state** - Use a clean/test account without clutter
6. **File size** - Keep images under 500KB (compress if needed)

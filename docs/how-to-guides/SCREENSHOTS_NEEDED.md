# Screenshots Needed for How-To Guides

All previous screenshots have been cleared. Capture new screenshots and save them with the **exact filenames** below. Paths are relative to this folder (`docs/how-to-guides/`).

**Save locations:**
- **Mobile app:** `images/screenshots/mobile-app/`

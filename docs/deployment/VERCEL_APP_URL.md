# Vercel app URL: oxford-house-mileage-tracker.vercel.app

The web portal is intended to be available at **https://oxford-house-mileage-tracker.vercel.app**.

## How to get this URL

The `*.vercel.app` URL is determined by your **Vercel project name**.

1. Open [Vercel Dashboard](https://vercel.com/dashboard) and select the project that deploys the admin-web frontend.
2. Go to **Settings → General**.
3. Under **Project Name**, change it to **oxford-house-mileage-tracker** (no spaces; hyphens are fine).
4. Save. Your main deployment will then be at **https://oxford-house-mileage-tracker.vercel.app**.

Preview deployments (e.g. from branches) may still use other URLs; the backend CORS allows all `*.vercel.app` origins.

## After renaming

- Update any bookmarks or links to use **https://oxford-house-mileage-tracker.vercel.app**.
- The previous URL (**oxford-mileage-tracker.vercel.app**) may redirect or stop working after the rename, depending on Vercel’s behavior.
- Backend CORS and config in this repo already allow both the old and new URLs.

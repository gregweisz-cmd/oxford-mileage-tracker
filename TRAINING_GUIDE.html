Quick Start Guide

For CEO/CFO: Viewing Organizational Overview

  1. Access Reports & Analytics Tab
    • Navigate to Finance Portal or Admin Portal
    • Click "Reports & Analytics" tab
  2. View Summary Dashboard
    • Review summary cards (Total Expenses, Active Reports, etc.)
    • Check submission funnel (visual representation of report statuses)
    • Review top cost centers by spend and miles
  3. Analyze Trends
    • Scroll to "Trends" section
    • View expense trend chart over time
    • Review forecast for next month
  4. Generate Custom Report
    • Scroll to "Report Builder" section
    • Select desired columns (e.g., Employee, Cost Center, Amount, Status)
    • Set date range (e.g., This Quarter, This Year)
    • Optionally filter by cost center or status
    • Click "Run Report" to view data
    • Click "Export to CSV" to download
  5. Schedule Automated Reports
    • Configure a scheduled report (e.g., Weekly Expense Summary)
    • Set frequency (Daily/Weekly/Monthly)
    • Add recipient email addresses
    • Select columns and filters
    • System will automatically email reports on schedule

Quick Reference

Status Definitions

Status Description
Draft Employee is still working on the report, not submitted
Submitted Employee submitted the report, awaiting supervisor approval
Pending Supervisor Report is waiting for supervisor approval
Pending Finance Report approved by supervisor, awaiting finance review
Approved Report has been fully approved and is ready for payment
Needs Revision Report was sent back to employee for corrections

Common Workflows

Employee Submits Report â†’ Supervisor Approves â†’ Finance Approves â†’ Payment

  1. Employee creates and submits expense report
  2. Supervisor receives notification
  3. Supervisor reviews and approves (or requests changes)
  4. If approved, report moves to Finance
  5. Finance reviews and approves
  6. Report is marked approved and ready for payment processing

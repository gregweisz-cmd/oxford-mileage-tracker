Need help with the Expense Tracker?

This page is for Oxford House employees, supervisors, and finance staff using the Oxford House Expense Tracker mobile app and web portal.

Common reasons to contact support

  • Problems signing in or resetting your password
  • Questions about mileage, receipts, or per diem entries
  • Issues submitting or approving an expense report
  • Errors or unexpected behavior in the mobile app or web portal

What to include in your request

When you contact support, please include as much detail as possible so we can help quickly:

  • Your full name and Oxford House email address
  • Whether you are using the mobile app or the web portal
  • The browser or device you are using (for example, “iPhone 14 / iOS 18” or “Chrome on Windows 11”)
  • A short description of what you were trying to do and what happened instead
  • Any error message you saw, and (if possible) a screenshot of the problem
